# Exemplar Profiles

The Taste Bank uses exemplar profiles as taste anchors, not templates.

Repo-owner agents should borrow the bar, not the build:

- product density,
- workflow clarity,
- demo compression,
- repo seriousness,
- artifact quality,
- evaluation rigor,
- anti-toy discipline.

Agents must not copy product ideas, branding, UI layouts, or implementation details from reference projects.
