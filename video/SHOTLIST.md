# Shotlist: More Code Is Not Progress

Audio bed: subtle procedural ambient music underneath all scenes, mixed well below Kokoro-ONNX narration.

## 0:00-0:12 — Cold Open

- Fast fragments show the portfolio scale: 20 candidates, repo owners, command receipts, proof-now finalists, and no winner.
- Kinetic thesis text: "More code is not progress."
- End on "Better evidence is progress."

## 0:12-0:35 — The Bottleneck

- Left stream: LOC, docs, demos, polish.
- Right stream: customer evidence, technical credibility, investment confidence.
- The "activity" stream fades behind the "evidence" stream.

## 0:35-1:05 — Control Plane

- Animated pipeline:
  `Product Repos -> Owner Agents -> Validator Gates -> Buyer Judges -> Compute Allocator -> Proof Prioritizer -> IC Decision`
- Evidence pulse travels through the line.
- Labels for `ready`, `compelling`, and `investable`.

## 1:05-1:35 — Portfolio Walkthrough

- Portfolio wall with the four tracks:
  - Finance
  - Founder tools
  - Education
  - Enterprise agents
- Each track panel names representative repos.
- Side panel highlights the five proof-now finalists.

## 1:35-2:00 — Portfolio Result

- Scoreboard rows animate in:
  - `20/20 ready`
  - `19/20 buyer-compelling`
  - `5 proof-now finalists`
  - `0/20 investable`
  - `No winner awarded`
- Final row uses restrained red emphasis.

## 2:00-2:25 — Evidence Gate

- Terminal contract:
  - `make demo`
  - `make test`
  - `make eval`
  - `make receipt`
- Three evidence definitions:
  - `ready = structurally judgeable`
  - `compelling = synthetic buyer usefulness`
  - `investable = external proof exists`

## 2:25-2:45 — The Discipline

- State machine collapses toward `external-proof-required` and `no winner`.
- Final system statement resolves next to the state machine.
- Final line:
  "More code is not progress. Better evidence is progress."
