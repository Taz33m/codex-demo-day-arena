# Top Repos To Revisit

Only these five repos should be reopened first, and only with external proof.

## 1. CanonKit

Worth revisiting because founder launch materials are easy to obtain, the output is concrete, and the claim ledger creates a sharp quality bar.

Reopen only if a real founder provides messy launch notes, proof claims, and draft positioning, then agrees to judge whether the generated launch pack is usable.

## 2. Wedgewise

Worth revisiting because founders often have messy customer interviews and immediate need for ICP clarity.

Reopen only if a founder provides 3-5 real or anonymized customer interviews and wants to judge whether the generated ICP memo changes positioning or customer discovery.

## 3. ProofPulse

Worth revisiting because investor updates are frequent, high-leverage, and easy for founders to recognize as useful or not.

Reopen only if a founder provides a real update draft plus receipts and agrees to judge whether the output is sendable with light edits.

## 4. PrepPilot

Worth revisiting because tutors can quickly evaluate whether a generated prep packet saves real session-prep time.

Reopen only if a tutor provides an anonymized worksheet, missed questions, or session notes and agrees to mark what they would keep, rewrite, or discard.

## 5. SignalDraft

Worth revisiting because public earnings data can produce a real-data proof run without waiting for proprietary files.

Reopen only if a finance operator reviews a memo generated from a real public earnings event and says it would save time, improve judgment, or enter their workflow.
