# Limitations

The arena is not proof that any generated candidate is a venture-scale startup.

## What It Can Prove

- product maturity,
- code and demo quality,
- artifact usefulness,
- evaluation discipline,
- portfolio-level compute allocation.

## What It Cannot Prove Alone

- real willingness to pay,
- true buyer urgency,
- retention,
- distribution,
- data availability,
- procurement feasibility,
- venture-scale demand.

Synthetic validation is useful, but it cannot replace real customer evidence.
