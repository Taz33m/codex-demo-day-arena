# Reproducibility

The public repository is a control-plane artifact. The 20 evaluated product repos are private, so the paper should be read as a systems case study rather than a fully reproducible benchmark.

## Publicly Reproducible

The public repo includes:

- control-plane scripts,
- policies and rubrics,
- prompts,
- sanitized example outputs,
- aggregate final reports,
- paper figures and tables.

A reader can inspect how the workflow is structured, how validator and allocation decisions are represented, and how the system distinguishes structural readiness, buyer usefulness, and investment-grade proof.

## Not Publicly Reproducible

The private product repos contain the candidate implementations and full run artifacts. They are excluded because the public artifact is intended to showcase the control plane, not publish raw generated product candidates.

## Claim Boundary

Public claims are limited to:

- aggregate portfolio outcomes,
- sanitized control-plane examples,
- policy and script behavior,
- documented limitations.

The repo does not claim real market validation, real buyer adoption, or investment-grade proof for any candidate.
